To install and run:
Using XAMPP and phpMyAdmin:
1. start MySQL and Apache from XAMPP.
2. In XAMPP, click 'admin' in mySQL column.
3. Create a new database named db2, keep default settings.
4. run the sql file named DB2.sql on your db2 database.
	-This creates all the tables and runs the insert statements.
	-You could also just copy the contents of DB2.sql into the SQL tab in phpMyAdmin
5. Open 'http://localhost/dashboard/DB2/SignIn.php' in your browser.

6. You can sign in or register as a new user.
	-Can sign in as a dummy user by viewing login information in the users table and picking a student.
